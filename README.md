# Promo Publisher

Сервис для автоматизированной публикации сообщений от пользовательского
Telegram-аккаунта в заранее разрешённых беседах.

Проект использует Telegram MTProto API через Telethon.

## Возможности

- авторизация пользовательского Telegram-аккаунта;
- белый список разрешённых получателей;
- несколько вариантов рекламных сообщений;
- отключение отдельных бесед;
- ротация сообщений;
- хранение состояния публикаций;
- дальнейшая поддержка расписания и ограничений по частоте.

## Структура

- `app/main.py` — запуск приложения;
- `app/auth.py` — авторизация Telegram-аккаунта;
- `app/publisher.py` — публикация сообщений;
- `app/chats.py` — конфигурация получателей;
- `app/messages.py` — работа с сообщениями;
- `app/discover_chats.py` — просмотр доступных диалогов;
- `data/chats.json` — список разрешённых получателей;
- `data/messages.json` — варианты сообщений.
